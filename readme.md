# 单元剧小说生成器 (Unit Novel Generator) 🚀

这是一个基于 FastAPI 和大模型 API 构建的**小说辅助创作工具**，专门为“单元剧”类型的小说量身定制。

## 📖 什么是单元剧？
在本项目中，单元剧指的是：每一个大章之间的关系并不大，主角在特定的背景下推进一条主线，但在推进的过程中，各个支线发展开来又彼此独立。

为了更直观地理解，你可以参考以下四部作品的叙事结构：
*   《奇诺之旅》（Kino no Tabi）
*   《少女终末旅行》（Shoujo Shuumatsu Ryokou）
*   《魔女之旅》（Majo no Tabitabi）
*   《虫师》（Mushishi）

## ⚙️ 核心板块设定 (以《少女终末旅行》为例)

如果你想用本生成器写出《少女终末旅行》这样的故事，你可以这样配置各个板块：

### 【初级板块】
*   **世界观设定**：“世界大战之后”的废土世界。
*   **人物**：两位主角。千户（喜欢读书，理性）和尤莉（喜欢吃，乐天）。*(注：新建人物需阐明性格、能力、人际关系及正反派定位)*
*   **主线**：“往上层城市走”，寻找生存希望。
*   **故事明面设定**：世界大战过后，几乎没有人了。战争遗留物很危险，人类在地表之上修建了多层结构。
*   **故事暗面设定（AI遵循但不直白点出）**：除了她们两人外，人类已逐渐灭绝。她们无法逃离命运，最终会在最高层冻死。
*   **故事主题风格**：诙谐、幽默与反战。
*   **文风选择**：带有哀伤但温暖的末日感。*(支持直接描述、仿写特定作者，或上传语料让 AI 提取文风)*
*   **伏笔**：随处可见的白色奇怪雕像（其实是新人类）。*(带有专门的伏笔收纳库，可手动或由 AI 在指定章节后自动填坑)*

### 【高级板块】
*   **内涵设定**：对战争的无声批判。
*   **输出要求**：所有人物对话必须用“……6……”代替，真实对白由用户后续自行填补。
*   **设定冲突检查**：AI 会自动纠错，例如“千户最开始设定是女生，后续如果生成了‘他’，则触发冲突提醒”。
*   **人物成长设定**：定义角色随剧情的缓慢转变（例如：小千在 15 章的跨度内，逐渐对读书失去兴趣）。
*   **AI 模型选择**：支持切换不同大模型内核（当前默认接入智谱 AI）。

## 🚀 本地运行指南
1. 安装依赖：`pip install -r requirements.txt`
2. 配置环境变量：在根目录创建 `.env` 文件，写入 `ZHIPU_API_KEY=你的密钥`
3. 启动后端服务：`python main.py`
4. 访问生成器：在浏览器打开 `http://127.0.0.1:8000`
